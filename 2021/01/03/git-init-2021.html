This is the first of a series of mini-posts of various computer housekeeping tips for my own use. Hopefully these may also be some some benefit for others.

Source control with git serves several functions:

  • It provides a method of keeping track of changes made to your code, so you can roll back to a previous save if needed.
  • If used with a remote hosting service such as GitHub, gives a convenient back up of your code in the cloud.
  • Allows easy collaboration with other developers.
  • Can be used by many providers for managing deployment.

Let’s get started:

Create Xcode Project - start with appropriate project

Navigate to your project folder using the command line. In Terminal put project under source control git init git add . git commit -m “Project setup”

Add a .gitignore

Create new repository on GitHub
